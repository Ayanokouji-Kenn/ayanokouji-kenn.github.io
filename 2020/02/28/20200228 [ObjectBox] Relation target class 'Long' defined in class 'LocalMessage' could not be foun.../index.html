错误[ObjectBox] Relation target class 'Long' defined in class 'LocalMessage' could not be foun...

发表于2020-02-28|更新于2021-03-10|Android
|阅读量:

使用ObjectBox时遇到的迷之问题，看到这个提示，怎么检查那个类都是加了@Entity注解的。
发生原因是：类里定义了ObjectBox不能理解的字段，比如var atIds: MutableList<Long> = mutableListOf()，要写个转换类加上去

法预料返回的是个什么，那不妨就从LinkedTreeMap中直接取数据。
